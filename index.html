Guia Rapida de Comandos Git

git init

Inicializa un nuevo repositrio git Locoal en el directorio actual.

git add .

Agrega todos los archivos del area de preparacion (staging)

git push

Envia los commits loacales al repositrio remoto

git status

Muestra el estado actual del repositorio archivos modificados y no rastreados

git commit -m "Mensaje "

Guarda los cambios agregados con un mensahe descriptivo

git pull

Obtiene y fusiona los cambios del repositrio remoto
